- readAsText( file, encoding ): to read file or blob with specified encoding

- readAsBinaryString( file ): Gets the raw binary data from any file or blob

- readAsArrayBuffer( file ): Generates an ArrayBuffer via reading a file and blob object.It returns
        Int8,Uint8 and byteLength arrays.
        This is helpfull when converting between extensions
